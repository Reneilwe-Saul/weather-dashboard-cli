# Weather Dashboard CLI 🌤

![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)
![API](https://img.shields.io/badge/OpenWeather_API-orange?style=flat)
![Status](https://img.shields.io/badge/Status-Complete-green)

A Python command-line app that fetches real-time weather data 
for any city in the world using the OpenWeather API.

---

## 📌 Problem Statement

Weather information is often buried in apps and websites. 
This tool gives instant, clean weather data directly in 
the terminal — no browser needed.

---

## 🔍 What It Shows

- 🌡 Temperature (°C)
- ☁️ Weather condition
- 💧 Humidity (%)
- 💨 Wind speed (km/h)

---

## 🚀 How to Run

1. Clone the repo:
   git clone https://github.com/Reneilwe-Saul/weather-dashboard-cli

2. Install the required library:
   pip install requests

3. Add your free API key from openweathermap.org into weather.py

4. Run the app:
   python weather.py
